# modern-g7-32

Шаблон для оформления документов в соответствии с ГОСТ 7.32-2017. Он был создан для автоматизации рутинных процессов при работе с научными работами. Шаблон может быть полезен студентам вузов при оформлении лабораторных, курсовых и дипломных работ. Отличается от оригинала тем, что расчитан для англоязычных работ студентов магистерской программы "Науки о Данных".
